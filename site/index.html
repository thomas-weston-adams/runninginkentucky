› About this site

Running in Kentucky is a community-built directory for run clubs and races in the Lexington area. It's a work in progress — no ads, no accounts, just a free resource for local runners.

Run Club Schedule

Weekly runs and club info come from a shared Google Doc that anyone can edit. If your club is missing, meets at a different time, or has changed locations — open the doc and update it. Changes appear on this site within 6 hours automatically.

Race Listings

Races are pulled automatically from multiple regional sources and merged:

  • John's Run Walk Shop — local race calendar
  • RaceRise — Kentucky & regional races
  • Front Runners Lexington — Kentucky race list
  • UltraSignup — trail & ultra events

This is an independent community project — not affiliated with any club or race organization. More features are in progress. Have a suggestion or found something broken? Send a note.
